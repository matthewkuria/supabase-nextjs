# User Management System

A modern, secure user authentication and profile management system built with Next.js 15, Supabase, and Tailwind CSS.

## 🚀 Features

- **Authentication**
  - User registration with email/password
  - Secure login functionality
  - Session management
  - Email confirmation support

- **Profile Management**
  - Automatic profile creation on signup
  - Profile data management (full name, username, website, avatar)
  - Real-time profile updates
  - Row Level Security (RLS) for data protection

- **UI/UX**
  - Modern glassmorphism design
  - Responsive layout
  - Loading states and error handling
  - Form validation
  - Accessible components

## 🛠️ Tech Stack

- **Framework**: [Next.js 15](https://nextjs.org/) with App Router
- **Authentication**: [Supabase Auth](https://supabase.com/auth)
- **Database**: [Supabase PostgreSQL](https://supabase.com/database)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Language**: TypeScript
- **Security**: Row Level Security (RLS)

## 📋 Prerequisites

Before you begin, ensure you have the following installed:

- Node.js 18.17 or later
- npm, yarn, or pnpm
- A Supabase account (free tier works)
